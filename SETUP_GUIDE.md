# 🎮 VKStream CraftCards - Полное руководство

> Интерактивная игровая система для стримов VK Video Live с админ-панелью на React

## 🚀 Архитектура системы

```
┌─────────────────────────────────────────┐
│   React Admin Panel (Frontend)          │
│   http://localhost:5173                 │
└──────────────┬──────────────────────────┘
               │ HTTP + WebSocket
               ▼
┌─────────────────────────────────────────┐
│   FastAPI REST API                      │
│   http://localhost:8001                 │
│   - /api/users, /analytics, /logs, etc  │
│   - /ws/live (WebSocket)                │
└──────────────┬──────────────────────────┘
               │ Async
               ▼
┌─────────────────────────────────────────┐
│   SQLite Database (stream_game.db)      │
│   - Users, Inventory, Logs              │
└─────────────────────────────────────────┘
```

## 🔧 Установка и запуск

### Шаг 1️⃣: Подготовка бэкенда

```bash
# Перейдите в корневую папку проекта
cd VKStream_CraftCards

# Установите зависимости Python
pip install -r requirements.txt

# Активируйте виртуальное окружение
.venv\Scripts\activate  # На Windows
source .venv/bin/activate  # На Linux/Mac
```

### Шаг 2️⃣: Запуск REST API

В первом терминале запустите API сервер:

```bash
python api.py
```

✅ API доступен по адресу: `http://localhost:8001`

Проверьте подключение:
```bash
curl http://localhost:8001/api/users
```

### Шаг 3️⃣: Запуск фронтенда React

Во втором терминале:

```bash
# Перейдите в папку фронтенда
cd frontend

# Установите зависимости
npm install

# Запустите dev сервер
npm run dev
```

✅ Админ-панель доступна по адресу: `http://localhost:5173`

## 📱 Что получилось

### ✨ React Admin Panel включает:

#### 1. **Dashboard** 📊
- Общая статистика (игроки, боксы, AC)
- Круговая диаграмма распределения боксов
- Топ-10 лучших игроков
- Средние показатели активности

#### 2. **Управление игроками** 👥
- Список всех игроков в таблице
- Поиск по нику или VK ID
- Прямое редактирование: звёзды, ПА, AC
- Удаление игроков с подтверждением

#### 3. **Выдача боксов** 🎁 (⭐⭐⭐ основная функция)
```
Возможно:
- Выбрать пользователя из списка
- Выбрать тип: Стандартный или Элитный
- Для элитных: выбрать рарность (1, 2 или 3 уровень)
- Указать количество боксов (1-100)
- Получить результаты с информацией о картах
```

Пример использования:
```
Дать 5 элитных боксов рарности 2 пользователю "Player123"
→ Каждый бокс содержит карты 1-2 уровня
→ Автоматический мерж 2-в-1
→ AC начислены в баланс
```

#### 4. **Логирование событий** 📜
- История всех открытий боксов
- Фильтры: все, стандартные, элитные
- Real-time обновления (каждые 5 сек)
- Отслеживание редких карт и мержей

### API Эндпоинты

```
GET  /api/users                 # Список пользователей
GET  /api/users/{vk_id}         # Данные конкретного
POST /api/users/{vk_id}/update  # Обновить параметры
DELETE /api/users/{vk_id}       # Удалить

POST /api/users/{vk_id}/boxes   # ⭐ Выдать боксы
GET  /api/users/{vk_id}/inventory

GET  /api/analytics             # Dashboard данные
GET  /api/logs                  # История событий
GET  /api/stats/timeline        # Временная шкала

POST /api/stream/finish         # Завершить стрим
POST /api/stream/clear-all      # Очистить БД

WS   /ws/live                   # Real-time обновления
```

## 🎯 Использование BoxManager (выдача боксов)

### Сценарий 1: Испытание для новичка
```
1. Выберите нового игрока
2. Стандартный бокс, количество 1
3. Нажмите "Выдать боксы"
```

### Сценарий 2: Поощрение активного
```
1. Выберите активного игрока
2. Элитный бокс, рарность 3 (максимум), количество 3
3. Он получит редкие карты!
```

### Сценарий 3: Тестирование
```
1. Выберите любого (или создайте)
2. Дайте 10 стандартных боксов
3. Посмотрите на Dashboard как выросла статистика
```

## 📊 Рарность для элитных боксов

| Рарность | Уровни карт | Пояснение |
|---------|-----------|----------|
| **1** | 1 лвл | Базовые карты, низкие шансы на редкое |
| **2** | 1-2 лвл | Смешанный дроп, лучше чем обычный бокс |
| **3** | 1-3 лвл | На 80% даёт карты 3 уровня! |

## 🔄 Real-time обновления

- Когда один администратор добавляет боксы
- Другие администраторы в реальном времени видят обновления через WebSocket
- Логи обновляются каждые 5 секунд
- Аналитика обновляется каждые 30 секунд

## 🐛 Решение проблем

### "Connection refused на порту 8001"
```bash
# Убедитесь, что api.py запущен
python api.py

# Или явно укажите
python api.py --host 0.0.0.0 --port 8001
```

### "Cannot connect to API from frontend"
```bash
# Проверьте CORS в api.py - они уже разрешены все источники

# Или в vite.config.ts проверьте proxy конфиг
proxy: {
  '/api': 'http://localhost:8001'
}
```

### "npm modules missing"
```bash
cd frontend
npm install
```
