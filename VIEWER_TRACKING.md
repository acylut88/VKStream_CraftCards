# 👥 Viewer Tracking System - Трекинг зрителей

## 🎯 Что это

Система автоматического отслеживания зрителей на стриме с начислением AC бонусов и выдачей milestones.

---

## 💰 AC Бонусы за присутствие (прогрессивная система)

| Время на стриме | AC бонус | Частота |
|-----------------|----------|---------|
| 0-30 мин | +5 AC | каждые 5 минут |
| 30-60 мин | +10 AC | каждые 5 минут |
| 60-120 мин | +20 AC | каждые 5 минут |
| 120-180 мин | +30 AC | каждые 5 минут |
| 180+ мин | +50 AC | каждые 5 минут |

**Пример:** За 3 часа стрима зритель заработает ~1290 AC только за присутствие!

---

## 🎁 Milestones (одноразовые бонусы)

| Время | Бонус | Описание |
|-------|-------|----------|
| 15 мин | +1 стандартный бокс | Первая награда за активность |
| 30 мин | +3 AC | Дополнительный AC |
| 60 мин | +1 элитный бокс | Редкий дроп |
| 180 мин | +1 стандартный бокс<br>+1 элитный бокс<br>+1 PA заряд | **Бонус ветерана** |

---

## 🔄 Как работает

### Основной цикл (каждые 5 минут):

```
1. Загрузить текущих зрителей из VK Live API
2. Для каждого зрителя:
   ├→ Обновить vk_id маппинг (никнейм → vk_id)
   ├→ Посчитать время на стриме
   ├→ Начислить AC (по прогрессивной шкале)
   ├→ Проверить milestones (если новые → выдать)
   └→ Уведомить в ЛС о новых бонусах
3. Кто ушёл → сессия продолжается (не сбрасывать)
4. Кто новый → зарегистрировать в CraftCards
```

## 🎮 Команды зрителей

| Команда | Описание |
|---------|----------|
| `!время`, `!сессия` | Время на стриме + AC бонусы |
| `!бонусы` | Какие milestones достигнуты |

### Пример ответа на `!время`:
```

AC заработано: 345
Текущий бонус: +20 AC / 5 мин
Milestones: 15 мин, 30 мин, 60 мин

### Пример ответа на `!бонусы`:
```
🎁 Твои бонусы на стриме:

✅ 15 мин: 📦 +1 стандартный бокс
✅ 30 мин: 💰 +3 AC

## 📊 БД структура

### Таблица `viewer_sessions`:
```sql
vk_id           INTEGER PRIMARY KEY  -- VK ID зрителя
nickname        TEXT                 -- Текущий никнейм
session_start   TIMESTAMP            -- Когда зашёл на стрим
last_seen       TIMESTAMP            -- Последний раз видели
total_minutes   INTEGER DEFAULT 0    -- Всего минут
ac_earned       INTEGER DEFAULT 0    -- AC заработано
milestones      TEXT DEFAULT '[]'    -- JSON: [15, 30, 60, 180]
```

## ⚠️ Важно

- ✅ Сессии **не сбрасываются** когда зритель уходит
- ✅ При возвращении время **суммируется**
- ✅ AC начисляется **сразу** каждые 5 минут
- ✅ Milestones выдаются **один раз**
- ✅ Все уведомления **в ЛС** (не в общий чат)
