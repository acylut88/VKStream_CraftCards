# 🎁 Auto-Reward System - Логика работы

## Как это работает

**Сообщение от ChatBot:**
```
ChatBot: SkilloCrabs получает награду: LootBox - Standart за 1
```

**"за 1"** = зритель активировал награду 1 раз за 1 балл канала

**Бот вызывает API 1 раз** → API открывает 1 бокс

---

## Логика расчёта боксов

### Стандартные боксы

| Активация за стрим | box_num | Карт (формула: 4 + box_num * 2) |
|---------------------|---------|----------------------------------|
| 1-я активация | 1 | 4 + 1*2 = **6 карт** |
| 2-я активация | 2 | 4 + 2*2 = **8 карт** |
| 3-я активация | 3 | 4 + 3*2 = **10 карт** |
| ... | ... | ... |
| 12-я активация | 12 | 4 + 12*2 = **28 карт** |

**Максимум за стрим:** 12 стандартных боксов

---

### Элитные боксы

| Активация за стрим | box_num | Карт |
|---------------------|---------|------|
| 1-я активация | 1 | **6 карт** |
| 2-я активация | 2 | **8 карт** |
| 3-я активация | 3 | **10 карт** |

**Максимум за стрим:** 3 элитных бокса

---

## Сброс счётчиков

### Новый стрим = сброс box_num до 1

Для сброса счётчиков всех пользователей:

```bash
curl -X POST http://127.0.0.1:8001/api/stream/start-day
```

Или через админку React → кнопка **"Начать новый стрим"**

## Архитектура

```
ChatBot сообщение
    ↓
bot_Handler: handle_reward_notification()
    ↓
Парсим: nickname="SkilloCrabs", count=1, type="Standart"
    ↓
┌─────────────────────────────────────┐
│ Проверка регистрации                │
│ └→ Если нет → auto-register         │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ Цикл: range(count)                  │
│ └→ POST /api/users/{nick}/boxes     │
│    └→ main.py::process_lootbox()    │
│       └→ box_num = db.counter + 1   │
│       └→ cards = 4 + box_num * 2    │
└─────────────────────────────────────┘
    ↓
Результат в логи
```

---

## Команды для управления стримом

| Endpoint | Действие |
|----------|----------|
| `POST /api/stream/start-day` | Сбросить счётчики боксов ВСЕХ пользователей |
| `POST /api/stream/session/create` | Создать новую сессию стрима |
| `POST /api/stream/finish` | Завершить стрим + экспорт CSV |

---

## Важно

- ✅ Счётчик `box_num` хранится **в базе данных** для каждого пользователя
- ✅ Счётчик **сбрасывается** при старте нового стрима
- ✅ Бот **не считает** box_num - это делает API
- ✅ Одна активация награды = **один вызов API**

---

**Готово!** 🎉 Логика уже правильная - бот просто вызывает API, а API считает карты по формуле.
